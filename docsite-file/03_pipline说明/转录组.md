# 转录组测序
### **　流程介绍**
　　转录组测序的研究对象为特定细胞在某一功能状态下所能转录出来的所有RNA的总和，它主要利用新一代高通量测序技术，对物种或组织的RNA反转录而成的cDNA文库进行测序，并得到相关的RNA信息。其研究对象为特定细胞在某一功能状态下所能转录出来的所有RNA的总和，包括mRNA和非编码RNA。它是指用新一代高通量测序技术，对物种或组织的RNA反转录而成的cDNA文库进行测序，并得到相关的RNA信息。
　　目前转录组测序技术广泛应用于生物学研究、医学研究、临床研究中。

#### **案例解析**
#### **　研究背景**
　　 杨树作为重要的经济、生态树种，育种工作中培育出的三倍体杨树性状优良，具有杂种优势，其生长速率、抗性和材质等指标都超越二倍体亲本，但由于林木的遗传背景复杂，目前对三倍体杂种优势的分子机理研究较少。一系列的研究以杨树三倍体及其二倍体亲本为材料，利用RNA-seq分析三倍体杂种优势和多倍化的相关分子机理。
　　2014-2015年，高通量测序和生物信息团队“四连助攻”了北京林业大学杨树育种课题组，公布了4篇研究成果。利用RNA-seq和生物信息分析揭示了三倍体杂交杨树中等位基因差异表达、杂种与亲本间基因差异表达的机制, 来认识三倍体杨树杂种优势形成的分子机理。
