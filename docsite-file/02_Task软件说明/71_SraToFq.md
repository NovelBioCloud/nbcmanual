# SraToFq
　　sra(short reads archive)是NCBI 推出的存储高通量数据的格式，而数据分析所使用得多为fastq格式。SraToFq是将sra格式的数据转换成fastq格式。

https://trace.ncbi.nlm.nih.gov/Traces/sra/sra.cgi?view=software
